# 📚 Cadastro de Aluno com Endereço

Projeto simples desenvolvido em PHP para realizar o cadastro de alunos junto com suas informações de endereço.

---

## 🚀 Funcionalidades

* Cadastro de aluno
* Cadastro de endereço
* Envio de dados via formulário
* Processamento com PHP
* Interface simples e funcional

---

## 🛠️ Tecnologias utilizadas

* PHP
* HTML5
* CSS3

---

## 📂 Estrutura do projeto

```
cadastro-aluno/
│── index.php        # Formulário de cadastro
│── processa.php     # Processamento dos dados
```

---

## ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/"SEU-USUARIO"/cadastro-php-com-endereco.git
```

2. Coloque o projeto em um servidor local, como:

* XAMPP
* WAMP
* Laragon

3. Acesse no navegador:

```
http://localhost/cadastro-aluno
```

---

## 💡 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos básicos de desenvolvimento web utilizando PHP, como:

* Manipulação de formulários
* Envio de dados via método POST
* Organização de código
